quickpay Change Log
=====================

### 1.0.0 - 2015-08-21

* 支付接口支付宝、微信扫码支付（包含下单和预下单）
* 支付接口微信公众号支付
* 支付接口微信企业付款
* HTTP 接口、TCP UTF-8 接口、TCP GBK 接口
* 管理平台配置代理、集团商户、商户、微信、支付宝
* 管理平台交易查询、下载
* 管理平台统计报表（新系统可下载报表）
* 新旧系统迁移、数据迁移


### 0.9.1 - 2015-06-15

* 统一使用 JSON 作为配置文件


### 0.9.0 - 2015-03-28

* 生产环境发布的第一个版本，支持绑定支付、ApplePay、无卡直接支付
