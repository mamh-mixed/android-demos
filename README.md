收银宝
=========


INSTALL
=======

```
# 通过 bower 安装前端依赖
cd static
bower install
cd ..

```

免翻墙安装golang.org/x/tools包
=======
```
git clone https://github.com/golang/tools $GOPATH/golang.org/x/tools 
cd $GOPATH
go install golang.org/x/tools/cmd/cover
```
